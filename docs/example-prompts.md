# Example prompts

These are ready-to-use prompts for testing MUSE. Just copy one and send it to your connected assistant — they go from quick checks to bigger tasks, so you can try things step by step.

> Tip: open a level in Helix first. You can always rephrase these in your own words.

## 1. Check the connection

- **"Are you connected to the Helix editor? Tell me the current project and level."**
  Confirms MUSE is working.
- **"List all the actors in the current level."**
  Shows that the assistant can read your scene.

## 2. Look around

- **"Pick any object in the level and tell me its position, size and components."**
  Reads details from the editor.
- **"Take a screenshot of the viewport so you can see the scene."**
  The assistant captures what's on screen.
- **"Point the camera at the player start and show me the view."**
  Moves the camera, then captures it.

## 3. Read the Helix documentation

- **"Using the Helix docs, explain how scripting events work."**
  Looks the answer up in the official documentation.
- **"Find the Helix docs page about custom vehicles and summarize the steps."**
  Searches and reads a page for you.
- **"What can the hCharacter class do? Check the Helix documentation."**
  Pulls API details straight from the docs.

## 4. Make a change

- **"Add a point light 300 units above the player start."**
  Places a light in the level.
- **"Spawn a cube at the world origin and name it TestCube."**
  Creates an object you can experiment with.
- **"Move TestCube up by 200 units, then take a screenshot."**
  Edits an object and shows the result.
- **"Duplicate TestCube three times, 150 units apart."**
  Repeats an object across the scene.
- **"Delete TestCube."**
  Cleans up when you're done.

## 5. Set the mood

- **"Add a warm directional light and adjust the sky to look like sunset."**
  Lighting and atmosphere in one go.
- **"Add some fog to the level to make it feel moody."**
  Quick visual effect.

## 6. Bigger tasks

- **"Build a small lit room: a floor, four walls, a ceiling, and a light in the middle. Show me a screenshot when it's done."**
  Combines several steps into one request.
- **"Read the Helix docs on the Character Creator, then tell me how I'd add a custom hairstyle."**
  Mixes documentation with a real how-to.

---

If something doesn't look right, just tell the assistant what you wanted instead — it can check the editor and fix it.
