# Tools

Once connected, your assistant can use the tools below to work inside Helix. You don't use them directly — you just ask for what you want, and the assistant picks the right tool for the job.

## Understand your project

* **inspect** — look at any object, actor, asset or level and read its details.
* **helix\_docs** — read the official Helix documentation to answer "how do I…" questions.

## Build your world

* **control\_actor** — add, move, rotate, duplicate or delete objects in the level.
* **manage\_level** / **manage\_level\_structure** — work with levels and how they're organized.
* **build\_environment** — set up environments and surroundings.
* **manage\_geometry** — create and edit shapes and meshes.
* **manage\_asset** — find, create and organize the things in your project.

## Control the editor

* **control\_editor** — move the camera, play and stop, run commands, take screenshots.
* **system\_control** — quality settings, performance checks and other editor tasks.
* **manage\_tools** — turn individual MUSE tools on or off.

## Look and sound

* **manage\_effect** — particles and visual effects.
* **manage\_audio** — sounds and audio.
* **manage\_sequence** — cinematics and sequences.
* **animation\_physics** — animation and physics.

## Gameplay

* **manage\_blueprint** — create and edit Blueprints.
* **manage\_character** — characters and the Character Creator.
* **manage\_ai** — AI behavior.
* **manage\_combat** — combat.
* **manage\_gas** — abilities.
* **manage\_interaction** — interactive objects.
* **manage\_inventory** — inventories and items.
* **manage\_networking** — multiplayer and networking.

_MUSE includes 200+ individual actions across these tools, and more are added over time._
